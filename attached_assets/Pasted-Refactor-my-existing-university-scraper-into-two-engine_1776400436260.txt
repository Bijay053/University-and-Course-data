ARCHITECTURE GOAL
==================================================

Create two scraping engines:

A. Fast Static Scraper
Use when:
- a clean course listing page is found
- number of course links is reasonably small, for example 1 to 30
- course pages are normal HTML
- key fields can be extracted from static HTML
- no major JavaScript interaction is needed for most pages

B. Advanced Smart Scraper
Use when:
- homepage must be researched to find listing pages
- listing is paginated or huge
- content is JS rendered
- course data is hidden behind tabs, accordions, toggles, APIs
- fees, English requirements, academic requirements, or campuses are on separate pages
- browser interaction is required for many pages

==================================================

Example helper:

```ts
function shouldUseFastStaticScraper(params: {
  listingUrl?: string | null;
  listingLinks?: string[];
  sampleStaticSuccessCount?: number;
  sampleCount?: number;
  siteLooksStatic?: boolean;
}) {
  const linkCount = params.listingLinks?.length || 0;
  const sampleSuccess = params.sampleStaticSuccessCount || 0;
  const sampleCount = params.sampleCount || 0;

  return Boolean(
    params.listingUrl &&
    linkCount > 0 &&
    linkCount <= 30 &&
    params.siteLooksStatic &&
    (
      sampleCount === 0 ||
      sampleSuccess >= Math.max(1, Math.floor(sampleCount * 0.6))
    )
  );
}
