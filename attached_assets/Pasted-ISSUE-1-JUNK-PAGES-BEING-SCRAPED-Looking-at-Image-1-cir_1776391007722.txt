THE TRUTH: STATIC HTML EXTRACTION CANNOT SOLVE VIT
You asked for static HTML only, but VIT's structure makes this impossible:
Why Static HTML Fails:

Domestic/International Toggle - JavaScript-based, changes content dynamically
Collapsed Accordions - Content hidden until clicked
Tab Navigation - Content loaded/shown on click

What static HTML sees:
html<!-- Default view (Domestic) -->
<div class="fee-display">$28,000</div>

<!-- International view (HIDDEN in static HTML) -->
<div class="fee-display" style="display:none;">$48,000</div>
The scraper cannot access the hidden content without JavaScript execution!

THE ONLY SOLUTION: BROWSER AUTOMATION
Here's what needs to happen for VIT (and similar universities):
Step 1: Click "International" Toggle
typescriptawait page.click('[data-student-type="international"]');
await page.waitForTimeout(2000); // Wait for content update
Step 2: Navigate to "Entry Requirements" Tab
typescriptawait page.click('text="Entry Requirements"');
await page.waitForTimeout(1000);
Step 3: Expand "Minimum English Language Requirement"
typescriptawait page.click('text="Minimum English Language Requirement"');
await page.waitForTimeout(500);
Step 4: Extract Data
typescriptconst html = await page.content();
