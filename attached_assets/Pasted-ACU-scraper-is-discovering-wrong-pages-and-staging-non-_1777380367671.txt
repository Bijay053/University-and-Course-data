Discovery issue

The crawler is picking research, supervisor, admission pathway and marketing pages as course candidates. It should only include URLs under real course detail pattern:

/course/{course-slug}

Not:
/research-and-enterprise/...
/international-students/english-and-pathway-programs/...
/study-at-acu/admission-pathways/...

Fee issue

Fee was extracted for the online course, but since this course is online only, it should not be staged as on campus.

Required fixes

Add stronger validation before staging:

must have a proper course title
must be individual course page, not category page
must not have title suffix from browser title
mode must come from course detail section only
do not use global campus list as location
online only courses should be skipped or marked online, not saved as on campus

Main issue:
ACU scraper is scraping navigation and marketing pages, then using global ACU campus names to fill missing locations. This creates fake on campus courses.
